## Zingy Quiz

Zingy Quiz is a quiz application to test your knowledge.

## Application Links

[Zingy Quiz Live](https://zingy-quiz-app.netlify.app/)

## Features

### P0

- Home Page
- Quizzes by Category
- Rules Page for while attempting Quiz
- MCQ based Quiz Question Page
- Result Page with Answers & Score

### P1

- Authentication
  - Sign-up
  - Login
  - Logout
- User Profile Page
- Quit Quiz
- Loading & Alerts
